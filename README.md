# App Clínica - Expo + SQLite

## Como rodar
1. Instale dependências:
   ```bash
   expo install expo-sqlite @react-navigation/native @react-navigation/native-stack react-native-screens react-native-safe-area-context react-native-gesture-handler
   ```
2. Rode o app:
   ```bash
   expo start
   ```
